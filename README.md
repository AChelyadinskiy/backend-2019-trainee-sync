# Pitter Sync

### Sync service for Pitter project


### Usage

- `make up` - Run in docker-compose
- `make down` - Stop and remove docker-compose services
- `make lint` - Run linter
- `make format` - Run auto-formatter for src/
- `make test` - Run tests
